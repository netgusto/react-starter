# React Starter

Starter repo for react+webpack+babel+eslint+geiger.

## Install & run

```
git clone https://github.com/netgusto/react-starter my-project-name
cd my-project-name
npm install
grunt serve
```

Then navigate to [http://localhost:8000]()

## Build

```
grunt build
```
